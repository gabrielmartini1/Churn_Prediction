# Customer Churn Prediction
An exploratory analysis of customer churn prediction data
<br><br><br>

<p align="center"><strong>RESUMO</strong></p>
 
<p align="justify"> Este estudo aborda a previsão de rotatividade de clientes em empresas de telecomunicações, analisando a aplicação de algoritmos de Machine Learning para identificar quais clientes estão mais propensos a cancelar seus serviços. Além disso, esse projeto discute o impacto da rotatividade de clientes no setor como um todo, bem como a importância das métricas CAC e LTV na avaliação da efetividade das estratégias de retenção de clientes. Por fim, o estudo também compara a eficácia de diferentes algoritmos de Machine Learning na previsão da rotatividade de clientes.</p>

<p align="justify">O objetivo geral deste estudo é explorar a aplicação de algoritmos de Machine Learning na previsão da rotatividade de clientes em empresas de telecomunicações. A rotatividade de clientes é um problema comum nesse setor, onde a competição é acirrada e a oferta de serviços similares é grande.</p>

**Palavras-Chave:** Machine Learning, Churn rate, predictive model.
